下面给你整理一份 **Cursor（默认设置）最实用、最高频、开发效率最高的快捷键清单**。
这份清单是按照 **日常开发场景** 分类整理，让你真正能记住、用得上。

所有快捷键均为 **Windows 默认设置**（如果你使用的是 Mac，我也可以给你 Mac 版）。

---

# 🧭 **一、Cursor 核心 AI 功能快捷键（最常用）**

### **1. 打开 Command Palette（AI 指令面板）**

```
Ctrl + Shift + P
```

使用频率最高。

作用：

* 运行 AI 指令
* 启动增量编辑
* 运行代码动作
* 打开设置、MCP 配置等

---

### **2. 打开 / 关闭 Chat 面板（AI 对话）**

```
Ctrl + I
```

这是 Cursor 的核心快捷键。

---

### **3. 启动智能补全（Inline Completion）**

```
Tab
```

当 Cursor 自动建议下一段代码时，按 Tab 接受。

---

### **4. AI 编辑选中代码（选区增强）**

选中一段代码 → 按：

```
Ctrl + K
```

会打开一个 AI 编辑输入框，可以输入：

* “优化这段代码”
* “加异常处理”
* “把这个函数改成异步”
* “转成 TypeScript”
* “为这个类加注释”

这是 Cursor 最强大的功能之一。

---

### **5. AI 修改本文件（文件级增量编辑）**

```
Ctrl + Shift + A
```

打开一个弹窗，让 AI 帮你：

* 重构整个文件
* 添加函数
* 改写逻辑

非常强大。

---

### **6. AI 运行解释 / 分析代码**

右键文件 → “Ask Cursor”
或快捷键：

```
Ctrl + I（打开 Chat）
→ 输入：/explain 或 Ask about this file
```

---

# 📁 **二、文件与工程操作快捷键**

### **1. 打开文件搜索**

```
Ctrl + P
```

输入文件名即可快速打开。

---

### **2. 全局搜索（搜索整个项目）**

```
Ctrl + Shift + F
```

---

### **3. 搜索并替换**

```
Ctrl + H
```

---

### **4. 关闭当前标签页**

```
Ctrl + W
```

---

### **5. 打开 Explorer（左侧文件栏）**

```
Ctrl + Shift + E
```

---

### **6. 打开终端**

```
Ctrl + `
```

（ESC 键下方的反引号）

---

# 🧱 **三、编辑器操作快捷键**

### **1. 跳转到定义（变量 / 函数 / 类）**

```
F12
```

---

### **2. 返回上一个位置**

```
Alt + ←
```

---

### **3. 前进**

```
Alt + →
```

---

### **4. 多光标编辑（非常好用）**

```
Alt + 点击
```

或：

```
Ctrl + Alt + ↑ / ↓
```

---

### **5. 复制当前行**

```
Shift + Alt + 下箭头
```

---

### **6. 向上 / 向下移动当前行**

```
Alt + ↑
Alt + ↓
```

---

### **7. 注释当前行**

```
// 单行
Ctrl + /

/* 多行 */
Shift + Alt + A
```

---

# 🛠 **四、Git 版本管理（内置功能）**

### **1. 打开 Source Control（Git 面板）**

```
Ctrl + Shift + G
```

---

### **2. 查看更改 diff**

点击 Git 面板中的文件即可。

---

### **3. 提交：**

```
Ctrl + Enter
```

（焦点在提交信息输入框里）

---

# 🤖 **五、Cursor 特有增强功能（AI+操作）**

### **1. 生成新文件（AI 写文件模板）**

按：

```
Ctrl + I → 输入：/new file
```

例如：

```
/new file 创建一个 fastapi 的用户路由
```

---

### **2. 生成整个项目结构**

```
Ctrl + I → /scaffold
```

例如：

```
/scaffold 生成一个 FastAPI + LangChain 后端项目
```

---

### **3. Ask Cursor About This File（分析文件）**

右键文件 → Ask Cursor
或：

```
Ctrl + I → 输入：/explain
```

---

### **4. 在代码中直接召唤 AI 补全（Ghost Text）**

默认开启，按 Tab 接受建议即可。

---
