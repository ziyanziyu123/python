# 一、30 天学习总纲（先稳住）

**学习对象不是“编程”，而是：**

> 👉 **如何构建一个“会思考 + 会行动 + 会记住”的系统**

```
第 1 周：编程与后端地基（不碰 Agent）
第 2 周：大模型 + Prompt（会用即可）
第 3 周：Agent 思维 + 状态设计（核心）
第 4 周：LangGraph + 工具调用 + 向量记忆（成型）
```

每天 **2–4 小时**，不需要全职。

---

# 二、30 天每日学习计划（可直接打卡）

你可以**直接复制到 Notion / Excel**，每天照着做。

---

## 🟦 第 1 周（Day 1–7）

### 目标：**不再害怕后端，理解 Agent 的“系统属性”**

---

### Day 1｜Python 入门（够用就停）

* 学：

  * 变量、print
  * if / for
* 目标：

  * 能运行 `.py` 文件
* 打卡标准：

  * 写 10 行不报错代码

---

### Day 2｜函数 & dict（非常重要）

* 学：

  * def
  * list / dict
* 理解：

  * dict = Agent 的状态容器

---

### Day 3｜typing & 类（只理解）

* 学：

  * class 是什么
  * TypedDict 是什么
* 目标：

  * 看懂状态定义代码

---

### Day 4｜FastAPI 入门

* 学：

  * 什么是 API
  * POST 请求
* 目标：

  * 启动一个接口

---

### Day 5｜JSON & 请求响应

* 学：

  * JSON 结构
* 目标：

  * 设计一个输入 / 输出 JSON

---

### Day 6｜数据库（只学 SQLite）

* 学：

  * 表 / 行 / 主键
* 目标：

  * 能存一条记录即可

---

### Day 7｜第 1 周复盘 ✅

你要能回答：

* Agent 为什么是后端系统？
* dict 在 Agent 里干嘛用？

---

## 🟦 第 2 周（Day 8–14）

### 目标：**会用大模型，但不迷信它**

---

### Day 8｜第一次调用大模型

* 学：

  * Prompt 是什么
  * temperature 是什么
* 目标：

  * 模型能回复你一句话

---

### Day 9｜结构化输出（关键）

* 学：

  * 为什么要 JSON 输出
* 练习：

```json
{ "summary": "", "confidence": 0.0 }
```

---

### Day 10｜模型一定会犯错

* 学：

  * 不稳定输出是常态
* 目标：

  * 接受 AI 不可靠

---

### Day 11｜Prompt ≠ Agent

* 学：

  * Prompt 的边界
* 思考：

  * 什么信息该进 state？

---

### Day 12｜LLM + Python

* 练习：

  * 模型输出 → 写入 dict

---

### Day 13｜“函数式 Agent”雏形

* 练习：

```python
state → step1 → step2
```

---

### Day 14｜第 2 周复盘 ✅

你要能说一句话：

> “Prompt 是工具，Agent 是系统。”

---

## 🟦 第 3 周（Day 15–21）

### 目标：**Agent 思维成型（最重要的一周）**

---

### Day 15｜什么是 Agent

* 理解：

  * Agent ≠ Chat
* 画流程：

```text
输入 → 分析 → 计划 → 行动 → 记忆
```

---

### Day 16｜State 是核心

* 学：

  * State = 唯一真相源
* 练习：

  * 设计 AgentState（dict）

---

### Day 17｜多步骤 & 回环

* 学：

  * 为什么必须拆步骤
* 练习：

  * step1 → step2 → step3

---

### Day 18｜多角色（概念）

* 理解：

  * 角色 ≠ 对话
  * 角色 = 职责函数

---

### Day 19｜失败、重试、人类介入

* 学：

  * Agent 一定会失败
* 思考：

  * 哪一步最容易错？

---

### Day 20｜Agent 的“记忆意识”

* 学：

  * 短期记忆 vs 长期记忆
* 不动手，只理解

---

### Day 21｜第 3 周复盘 ✅

你要能说清楚：

> “Agent 是一个有状态的流程系统。”

---

## 🟦 第 4 周（Day 22–30）

### 目标：**Agent 内核成型（会思考 + 会行动 + 会记住）**

---

### Day 22｜LangGraph 入门

* 学：

  * StateGraph
  * Node / Edge

---

### Day 23｜第一个 Agent Graph

* 练习：

  * 2–3 个节点跑通

---

### ⭐ Day 24｜工具调用（第一次）

* 学：

  * Tool = 函数
* 练习：

  * Agent 调用 `log_event()`

---

### ⭐ Day 25｜向量数据库（第一次）

* 用：

  * Chroma / 本地
* 目标：

```python
add(text)
search(query)
```

---

### ⭐ Day 26｜工具 + 向量库进 Agent

* 跑通：

```text
Agent → 写记忆 → 查记忆
```

---

### Day 27｜多角色 Agent

* 角色示例：

  * Reflector
  * Planner
  * Memory

---

### Day 28｜调试 Agent

* 学：

  * 日志
  * 回放
* 目标：

  * 知道 Agent 在哪一步错

---

### Day 29｜真实问题测试

* 用你自己的真实问题跑 Agent
* 看：

  * 是否重复
  * 是否失忆

---

### Day 30｜毕业日 🎓

你应该能自信地说：

> **“我做的是智能体系统，不是聊天机器人。”**
